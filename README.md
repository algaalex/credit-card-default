Credit risk prediction is an important task in financial institutions, as lenders must assess the probability that customers will fail to repay borrowed money. Accurate prediction of default risk helps financial institutions reduce losses and make informed lending decisions.           Traditional statistical techniques such as logistic regression have commonly been used for credit scoring; however, machine learning methods have demonstrated strong predictive capability in structured financial datasets. This project investigates whether machine     learning models can effectively predict credit card default behaviour using customer         financial history data.
The dataset used in this study is the Default of Credit Card Clients dataset obtained from the UCI Machine Learning Repository. The dataset contains 30,000 records and 25 variables    describing customer demographic information, credit limits, repayment behaviour, billing amounts, and payment history over six months. The target variable indicates whether a   customer defaulted on payment in the following month. Data preprocessing included    cleaning column names, handling categorical variables, feature engineering, and splitting the data into training and testing sets.
Three models were implemented: Logistic Regression as an interpretable baseline, Random Forest as an ensemble learning model, and XG Boost as a gradient boosting algorithm.      Hyperparameter tuning was applied to optimize model performance. Model evaluation was conducted using accuracy, precision, recall, F1-score, and ROC-AUC metrics.
Results show that the Tuned Random Forest model achieved the best performance with an accuracy of 0.814 and ROC-AUC of 0.781. Repayment delay variables were identified as the most important predictors of default risk. The results demonstrate that financial behaviour indicators are effective predictors of credit default and that machine learning models can support decision-making in financial risk assessment.
